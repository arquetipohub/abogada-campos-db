# Abogada de Campos

Repositorio de arquitectura de base de datos para el sistema jurídico del estudio.

Incluye estructura de:

- clientes
- casos
- bandeja de ingreso
- documentos
- trámites y presentaciones
- organismos y prestadores

Este repositorio servirá para exportar o recrear el sistema en:

- PostgreSQL
- Airtable
- Baserow
- NocoDB
- otros gestores de base de datos.
